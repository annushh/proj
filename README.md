# ML-инженер — профессия будущего

Это небольшой учебный сайт про профессию **ML-инженера**.  
Мы сделали его как понятную и визуальную страницу: кто такой ML-инженер, чем он занимается, как студенту войти в профессию и почему это направление важно для IT и бизнеса.

Проект  собран под ПК-версию
## Что есть на сайте

На странице есть несколько смысловых блоков:

- главный экран с кратким описанием профессии;
- блок «Кто такой ML-инженер?»;
- карточки с основными задачами ML-инженера;
- пошаговый план входа в профессию для студента;
- блок «Один день ML-инженера»;
- объяснение, почему ML полезен для ЭР-Телеком;
- интерактивные карточки;
- небольшой тест, который показывает, насколько профессия может подойти пользователю.

## Технологии

В проекте используются:

- **HTML** — структура страницы;
- **CSS** — стили, карточки, адаптив, анимации и оформление;
- **JavaScript** — плавная прокрутка, появление блоков при скролле, интерактивные карточки и тест;
- **Google Fonts** — шрифт Unbounded.

## Структура проекта

```text
project/
├── index.html      # основная HTML-страница
├── style.css       # стили сайта и мобильная адаптация
├── script.js       # интерактивность сайта
└── README.md       # описание проекта
```


## Адаптивность

Сайт рассчитан на разные размеры экрана:

- ПК и ноутбуки;

## Интерактивность

В `script.js` реализованы:

- плавный переход к блоку карьеры;
- появление секций при прокрутке;
- раскрытие интерактивных карточек на телефоне;
- выбор ответов в тесте;
- подсчет результата теста.

## Цель проекта

Цель проекта — показать профессию ML-инженера простым и понятным языком, а также сделать сайт аккуратным, современным и удобным не только на компьютере, но и на телефоне.
